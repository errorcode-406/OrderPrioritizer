# Order Prioritizer

Aplikasi Desktop Java untuk memprioritaskan order harian.

## Fitur
- Tambah order dengan ID, customer, dan prioritas (High/Medium/Low)
- Load dan save data order ke CSV
- Tampilan GUI modern menggunakan FlatLaf
- Tabel order dengan sorting sederhana

## Teknologi
- Java 17+
- Swing GUI
- FlatLaf Look and Feel
- CSV file storage

## Cara Jalankan
1. Buka project di NetBeans
2. Jalankan class `Main.java`
3. Tambah order dan simpan data
